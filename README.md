# 🌍 EnviroCare Application

## 📌 Overview
EnviroCare is a web application designed to monitor environmental risks and provide analytics dashboards.

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- REST APIs

## ✨ Features
- User Authentication
- Dashboard Analytics
- Risk Gauge Monitoring
- Alerts System
- Settings Management

## 🚀 How to Run
1. Clone the repo
2. npm install
3. npm run dev
